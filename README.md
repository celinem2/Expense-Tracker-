## Expense Tracker
A simple console-based Expense Tracker built in Python.  
This project allows users to add, view, filter, and total their expenses via the terminal.

## Features
- Add expenses (amount + category)
- View all expenses
- Filter by category
- Show total expenses
- Exit program gracefully
